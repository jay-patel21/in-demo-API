## Version 
Node 12

## Installation

npm install -g ts-node

npm install

## Commands 
### lint
npm run lint 

### start
npm start

## Database Setup

	a. Add credentials to ormConfig of the environment file.

	b. Create an empty database on the local system.

	c. Schema will be created on server start.
